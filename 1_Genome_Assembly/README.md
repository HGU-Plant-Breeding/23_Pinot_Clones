# Genome Assembly, Annotation, and Methylation Analysis

This directory contains the custom scripts and pipelines used for the structural evaluation, annotation, and epigenetic characterization of the Pinot Noir genome assembly of clone 20-13 Gm.

## Dependencies

To run these scripts, the following tools and libraries are required:

*   **Python 3.8+**
    *   Libraries: `pandas`, `numpy`, `scipy`, `pybedtools`, `argparse`, `intervaltree`
*   **Bioinformatics Tools**:
    *   `bedtools` (Required for most scripts)
    *   `samtools`
    *   `deepTools` (For matrix generation)
    *   `Mikado` & `TransDecoder` (For annotation pipeline)
    *   `Mummer` / `SyRI` (Upstream inputs)

---

## 1. Genome Structure & Classification

### `syri_classify_genome.py`
**Description:**  
Classifies the genome into **Hemizygous**, **Heterozygous**, and **Homozygous** regions based on SyRI structural variant output. It integrates small variant density to calculate these states.

**Usage:**
```bash
python syri_classify_genome.py \
    --syri-out syri.out \
    --chr-sizes genome.sizes \
    --output classified_genome.bed \
    --target-genome A
```

### `calc_genome_composition.py`
**Description:**
Calculates the genomic partition sizes (in bp and %) from a BED file (e.g., Homozygous vs. Hemizygous vs. Heterozygous).
**Usage:**

```Bash
python calc_genome_composition.py genomic_partitions.bed
```
## 2. Methylation Analysis
### GBM_calc.py
**Description:**
Performs Gene Body Methylation (GBM) classification. It uses a Binomial Test to classify genes as Body Methylated (BM), Intermediate (IM), or Unmethylated (UM) by comparing individual gene methylation against the global background (pCG).
**Usage:**

```Bash
python GBM_calc.py genes.bed cg_counts.bed output_classification.tsv --format counts
```
### calc_promoter_meth.py
**Description:**
Calculates average methylation statistics (Mean % and CpG count) per feature (e.g., Promoters). Expects a 10-column input generated by bedtools intersect -wa -wb.
**Usage:**

```Bash
python calc_promoter_meth.py intersected_promoters_methylation.tsv --output stats.tsv
```
### bin_methylation_diploid.sh
**Description:**
A Bash script that segregates methylation data into Low (<30%), Medium (30-70%), and High (>70%) bins. It processes diploid files and splits statistics by Haplotype and Feature (Genes/TEs).
**Usage:**

```Bash
./bin_methylation_diploid.sh ./input_data output_summary.csv
```
### deep_tools_matrix.sh
**Description:**
A SLURM wrapper for deepTools computeMatrix. It generates the matrix required to plot metaprofiles (e.g., methylation over gene bodies) for Heterozygous, Homozygous, and Hemizygous regions.
**Usage:**

``` Bash
sbatch deep_tools_matrix.sh CG genes
```
## 3. Comparative Genomics (Orthology)
### get_one_to_one_orthologs.py
**Description:**
Parses OrthoFinder output (Orthogroups.tsv) to extract high-confidence 1-to-1 orthologs between the two haplotypes.
**Usage:**

``` Bash
python get_one_to_one_orthologs.py Orthogroups.tsv --output 1_to_1_orthologs.tsv
```

### classify_gense_orthology.py
**Description:**
Classifies a specific list of genes based on their orthology context. It determines if a gene has an ortholog in the other haplotype or a paralog in the same haplotype.
**Usage:**

``` Bash
python classify_gense_orthology.py gene_list.txt Orthogroups.tsv
```
## 4. Transposable Elements (TEs)
### TE_divergence.sh
**Description:**
Processes EDTA annotations to analyze TE age.
Creates a master BED file with Divergence encoded in the ID.
Splits TEs into Young (<2% divergence) and Old (>20% divergence) sets for specific genomic partitions.
**Usage:**

``` Bash
./TE_divergence.sh --main genome.EDTA.gff3 --hetero het_tes.gff ...
```
## 5. Statistical Tools & Annotation
### run_permutation_test.py
**Description:**
Performs a Monte Carlo Permutation Test to assess the statistical significance of overlaps between two genomic feature sets, constrained to a specific background "universe" (e.g., checking if genes overlap TEs more than expected by chance).
**Usage:**

```Bash
python run_permutation_test.py query.bed db.bed universe.bed 10000
```
### mikado_pipeline.slurm
**Description:**
The SLURM submission script for the Mikado annotation pipeline. It consolidates transcript assemblies, runs BLAST/TransDecoder, and selects the best gene loci.

## 6. Visualization (plotsr_modified)
The folder plotsr_modified/ contains a customized version of the PlotSR tool, extended to visualize variant density heatmaps directly on chromosome bars. Please refer to the README.md inside that directory for installation and usage instructions.

# 1. Genome Assembly

Scripts for structural characterisation, annotation, and epigenomic analysis of the diploid Pinot noir reference genome (clone 20-13 Gm).

---

## Dependencies

**Python 3.8+:** `pandas`, `numpy`, `scipy`, `pybedtools`, `intervaltree`  
**Tools:** `bedtools`, `samtools`, `deepTools`, `SyRI`, `Minimap2`, `Mikado`, `TransDecoder`, `EDTA`, `OrthoFinder`

---

## 1. Genome Structure & Classification

### `syri_classify_genome.py`
Classifies the diploid genome into **Homozygous**, **Heterozygous**, and **Hemizygous** partitions based on SyRI output. Homozygous regions are defined as long syntenic blocks with near-zero sequence divergence (<5 SNPs/indels per 10 kb). Hemizygous regions encompass non-aligned regions, large structural insertions/deletions (>1kb), and highly diverged syntenic blocks. Heterozygous regions are syntenic blocks with high small-variant density.

```bash
python syri_classify_genome.py \
    --syri-out syri.out \
    --chr-sizes genome.sizes \
    --output classified_genome.bed \
    --target-genome A
```

### `calc_genome_composition.py`
Calculates the size (bp and %) of each genomic partition from the BED output of `syri_classify_genome.py`.

```bash
python calc_genome_composition.py genomic_partitions.bed
```

## 2. Methylation Analysis of the Reference Genome

### `bin_methylation_diploid.sh`
Segregates per-site methylation data from the reference clone into Low (<30%), Medium (30–70%), and High (>70%) bins. Processes both haplotypes and splits output by feature type (genes/TEs).

```bash
./bin_methylation_diploid.sh ./input_data output_summary.csv
```

### `deep_tools_matrix.sh`
SLURM wrapper for deepTools `computeMatrix`. Generates methylation metaprofile matrices over gene and TE bodies, stratified by genomic partition (Heterozygous, Homozygous, Hemizygous).

```bash
sbatch deep_tools_matrix.sh CG genes
```

### `calc_promoter_meth.py`
Calculates mean CG methylation and CpG count per promoter region (2kb upstream of TSS). Input is generated by `bedtools intersect -wa -wb`.

```bash
python calc_promoter_meth.py intersected_promoters_methylation.tsv --output stats.tsv
```

### `run_permutation_test.py`
Monte Carlo permutation test for the significance of overlap between two genomic feature sets within a defined background universe. Used to test whether asymmetrically methylated promoter pairs (AMPs) are enriched for large indels (≥50bp) compared to all heterozygous promoters.

```bash
python run_permutation_test.py query.bed database.bed universe.bed 10000
```

> **Note:** The indel size threshold used in the manuscript is ≥50bp. Ensure the `database.bed` file is pre-filtered accordingly.

## 3. Orthology & Gene Classification

### `get_one_to_one_orthologs.py`
Parses OrthoFinder output to extract high-confidence 1-to-1 ortholog pairs between the two haplotypes.

```bash
python get_one_to_one_orthologs.py Orthogroups.tsv --output 1_to_1_orthologs.tsv
```

### `classify_gense_orthology.py`
Classifies genes by their orthology context: whether they have a syntenic counterpart on the opposite haplotype or duplicated copies within the same haplotype.

```bash
python classify_gense_orthology.py gene_list.txt Orthogroups.tsv
```

---

## 4. Transposable Elements

### `TE_divergence.sh`
Processes EDTA TE annotations to estimate TE age via sequence divergence from the consensus. Splits TEs into Young (<2% divergence) and Old (>20% divergence) sets for each genomic partition.

```bash
./TE_divergence.sh --main genome.EDTA.gff3 --hetero het_tes.gff3 \
    --homo homo_tes.gff3 --hemi hemi_tes.gff3
```

---

## 5. Annotation Pipeline

### `mikado_pipeline.slurm`
SLURM submission script for the Mikado gene annotation pipeline. Integrates ab initio predictions (Helixer) with lifted-over reference gene models (Liftoff from PN40024.v4) and selects high-confidence gene loci.
